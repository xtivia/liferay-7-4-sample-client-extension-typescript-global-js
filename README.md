# liferay-7-4-sample-client-extension-typescript-global-js

A blueprint for a Global Javascript client extension with TypeScript/Webpack/ESLint for Liferay 7.4. 

This accompanies the instructional blog article [Creating a Liferay global JS client extension with TypeScript and Webpack](https://xtivia.com/blog/creating-a-liferay-global-js-client-extension-with-typescript-and-webpack).

## Getting Started

This should be in the client extensions folder of an active Liferay 7.4 workspace. Make sure your terminal is pointed to your workspace and enter the following:

```bash
blade gw clean deploy
```
